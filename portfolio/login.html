About AES
The Advanced Encryption Standard (AES) is a symmetric block cipher chosen by the U.S. government to protect classified information. AES is implemented in software and hardware throughout the world to encrypt sensitive data. It is essential for government computer security, cybersecurity and electronic data protection.

AES-128 uses a 128-bit key length to encrypt and decrypt a block of messages, while AES-192 uses a 192-bit key length and AES-256 a 256-bit key length to encrypt and decrypt messages. Each cipher encrypts and decrypts data in blocks of 128 bits using cryptographic keys of 128, 192 and 256 bits, respectively.

Implentation
When a string of information is parsed to be encrypted, using the AES 256 BIT version the string goes through 16 rounds of permutations and combinations and the result is such that the the string is now completely changed and cannot be of any use unless 1st decrypted.

These rounds are not completely random, they work on a key that is also used to decrypt the message, thus even if the same string goes through the encryption 2 times, the encrypted string is different but the decrypted one is always the inserted string.
